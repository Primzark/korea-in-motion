# Korea — Between Stillness and Signal

An image-led editorial site exploring Korea through living heritage, Seoul after dark, landscape, seasons, and the shared table.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

Built with React, Vite, Framer Motion, and Lucide icons. Motion respects the user's reduced-motion preference.
